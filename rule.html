    * 홍보

    서버의 공식적인 홍보 외에 불특정 다수에게 홍보하는 행위 금지.

    동화고등학교 학생이 아닌 사람들(동화중학교 포함)에게 서버를 홍보하거나 공유하는 행위 금지.

    (발견 시 빠른 제보 부탁드립니다.)

    서버 관련 홍보는 동화고등학교 학생들을 대상으로 한 홍보만 허용.



    [인게임]

    * 지옥문

    지옥문 생성 시 근처 128블록 이내에 이미 설치된 지옥문이 있는지 확인 후 생성 권고.

    128블록 이내에 지옥문이 여러 개일 경우 서로 간섭하여 꼬일 수 있음.
